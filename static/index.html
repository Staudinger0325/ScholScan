Abstract

ScholScan provides all the necessary materials for replicating experiments, including the dataset, precomputed embeddings, evaluation scripts, and inference pipelines. It supports both main results and retrieval-augmented generation (RAG) experiments across image and text modalities.

Due to storage constraints, large data artifacts may be hosted externally (e.g., Hugging Face). Please refer to the repository README for details.
